# Trang Web Đám Cưới - Minh Hảo & Phi Ta

Trang web landing page đám cưới với thiết kế thanh lịch và hiện đại.

## Cấu trúc thư mục

```
wedding_web/
├── wedding-website-exact-v2.html  # File HTML chính
├── css/                            # Thư mục chứa CSS
│   └── styles.css                 # File CSS chính
├── images/                         # Thư mục chứa ảnh
│   ├── README.md                  # Hướng dẫn về ảnh
│   ├── header-photo.jpg           # Ảnh cô dâu chú rể chính (cần thêm)
│   ├── gallery-1.jpg              # Ảnh gallery 1 (cần thêm)
│   ├── gallery-2.jpg              # Ảnh gallery 2 (cần thêm)
│   └── gallery-3.jpg              # Ảnh gallery 3 (cần thêm)
└── README.md                      # File này
```

## Hướng dẫn sử dụng

### 1. Thêm ảnh vào thư mục `images/`

Bạn cần thêm 4 ảnh sau vào thư mục `images/`:

- **header-photo.jpg** (hoặc .png)
  - Ảnh cô dâu chú rể chính ở phần header
  - Kích thước đề xuất: 125x165px (tỷ lệ dọc)
  
- **gallery-1.jpg** (hoặc .png)
  - Ảnh đầu tiên trong gallery
  - Kích thước đề xuất: 280x280px (vuông)
  - Ảnh cô dâu chú rể trong trang phục truyền thống Việt Nam
  
- **gallery-2.jpg** (hoặc .png)
  - Ảnh thứ hai trong gallery
  - Kích thước đề xuất: 280x280px (vuông)
  - Ảnh cô dâu chú rể trong trang phục cưới phương Tây
  
- **gallery-3.jpg** (hoặc .png)
  - Ảnh thứ ba trong gallery
  - Kích thước đề xuất: 280x280px (vuông)
  - Ảnh cô dâu chú rể trong trang phục cưới phương Tây

### 2. Mở trang web

Mở file `wedding-website-exact-v2.html` trong trình duyệt web để xem kết quả.

### 3. Tùy chỉnh

Bạn có thể chỉnh sửa nội dung trong file HTML:
- Tên cô dâu chú rể
- Thông tin đám cưới (ngày, giờ, địa điểm)
- Nội dung câu chuyện
- Câu hỏi FAQ
- Link xác nhận tham dự

## Tính năng

- ✅ Thiết kế responsive (tự động điều chỉnh trên mobile)
- ✅ Hiệu ứng fade-in khi scroll
- ✅ Hỗ trợ ảnh với fallback khi ảnh chưa được thêm
- ✅ Trang trí hoa lá SVG đẹp mắt
- ✅ Typography đẹp với Google Fonts

## Lưu ý

- Tất cả ảnh sẽ tự động được resize để phù hợp với layout
- Định dạng JPG hoặc PNG đều được hỗ trợ
- Nên sử dụng ảnh có chất lượng cao để hiển thị đẹp nhất
- Nếu ảnh chưa được thêm, trang web sẽ hiển thị placeholder với hướng dẫn

## Hỗ trợ trình duyệt

Trang web hoạt động tốt trên:
- Chrome (khuyến nghị)
- Firefox
- Safari
- Edge
